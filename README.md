Liar's Bar()
概要

物理的なトランプと模擬銃を使用して行う対面パーティゲーム「Liar's Bar（実写版）」の進行管理・演出補助を行うWebアプリケーションである。
ゲームの「お題（Table Card）」の抽選、ロシアンルーレットの確率管理、およびプレイヤーの生存管理を行う。

最大の特徴は Web Bluetooth API (BLE) を介した物理デバイスとの連携機能だ。ESP32等のマイコンを搭載した模擬銃のトリガー操作を検知し、ブラウザ上の発砲判定を自動で実行する。

特徴

ゲーム進行管理: ラウンドごとのお題（Q, K, A）のランダム選出。

ロシアンルーレット・ロジック:

6発中1発の実弾シミュレーション。

不発時のシリンダー回転（確率変動）処理。

発射時のシリンダーリセット処理。

BLEハードウェア連携: 物理コントローラーからの信号（Notify）を受信し、ゲーム内のトリガー処理を実行。

演出機能: 発砲音/空撃ち音の再生、画面シェイクなどの視覚効果。

レスポンシブUI: スマートフォンおよびPCでの操作に対応したダークテーマUI。

技術スタック

HTML5 / CSS3 / JavaScript (Vanilla): 依存関係を排除し、単一ファイル（および音声ファイル）のみで動作する軽量設計。

Web Bluetooth API: ブラウザとBLEデバイス間の通信に使用。
